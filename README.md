# Voice Calendar · 语音日历工具

> 以语音交互为核心、参考 Notion Calendar 设计风格的简约日程管理工具。

本仓库为 2026 年开发周期赛题作品仓库，按"PR 拆分、单 PR 单功能、持续交付"的规范进行开发。详细设计与功能拆解见后续 PR 中陆续完善的文档与代码。

## 仓库导航

仓库当前处于初始化阶段，后续将通过独立 PR 逐步引入：

- 项目脚手架（目录结构、.gitignore、根 README 详尽说明）
- 后端基础（FastAPI + SQLite + SQLAlchemy）
- 前端基础（Next.js 14 + TypeScript + Tailwind CSS）
- 数据模型与事件 CRUD API
- 日历多视图（日 / 周 / 月 / 年）
- 语音输入与指令解析
- 语音合成与播报
- 拖拽调整、快速创建、事件标签
- 用户认证、提醒、搜索、深色模式

每个 PR 均遵循"一次只做一件事"的原则，标题、功能描述、实现思路、测试方式齐全。

## 协作规范

- **分支策略**：所有功能在 `feat/<topic>` 分支开发，禁止直推 `main`。
- **提交节奏**：全周期持续提交，避免临尾突击。
- **依赖声明**：所有第三方库须在对应模块 README / requirements 内列明并标注用途。
- **可复现性**：`main` 分支任意时刻 clone 即可按 README 步骤运行。
